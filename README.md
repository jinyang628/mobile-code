# Mobile Code

Practice LeetCode problems on mobile.

## Tech Stack

- Mobile Frontend: Expo (React Native) with [React Native Reusables](https://rnr-docs.vercel.app/getting-started/introduction/)
- Server: FastAPI
- Hosting: [Render](https://render.com)
